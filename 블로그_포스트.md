# 나만의 운동 기록 웹앱 만들기

운동을 꾸준히 하려면 기록이 중요하다. 매일 어떤 운동을 했는지, 무게는 얼마나 들었는지 체크하면 성장하는 게 눈에 보인다. 그래서 직접 운동 기록 앱을 만들어봤다.

---

## 어떤 앱인가?

아이폰에서도 사용할 수 있는 웹 기반 운동 기록 앱이다. 홈 화면에 추가하면 네이티브 앱처럼 사용할 수 있다.

**주요 기능:**
- 요일별로 다른 운동 루틴 설정
- 무게와 세트 수 기록
- 완료한 운동 체크
- 진행률 실시간 표시
- 과거 기록 조회

---

## 화면 구성

### 메인 화면

상단에 주간 달력이 있고, 오늘 날짜가 강조 표시된다. 완료한 날은 초록색 체크 표시가 뜬다.

중간에는 오늘의 운동 목록이 나온다. 각 운동별로 무게(kg)와 세트 수를 입력할 수 있고, 완료하면 체크박스를 누르면 된다.

진행률 바가 실시간으로 업데이트되어서 얼마나 했는지 한눈에 보인다.

### 설정 화면

오른쪽 상단 ⚙️ 버튼을 누르면 요일별 운동을 편집할 수 있다.
- 운동 부위 이름 변경 (가슴, 등, 하체 등)
- 운동 종목 추가/삭제
- 휴식일 지정

---

|------|------|

| 토 | 팔 |
| 일 | 휴식 |

---

## 디자인

피트니스 앱답게 다크 모드를 기본으로 했다. 검정 배경에 오렌지색 포인트를 줘서 에너지 넘치는 느낌을 살렸다.

**색상 컨셉:**
- 배경: 블랙
- 강조: 오렌지 그라데이션
- 완료: 그린
- 삭제: 레드

버튼을 누르면 살짝 눌리는 애니메이션이 있어서 앱처럼 자연스럽다.

---

## 기술 스택

- HTML
- CSS (다크 테마, 반응형)
- JavaScript (바닐라)
- localStorage (데이터 저장)

별도의 서버 없이 브라우저에서만 동작한다. 데이터는 각 기기에 저장되기 때문에 인터넷 없이도 사용 가능하다.

---

## 사용법

1. 오늘 할 운동 확인
2. 운동하면서 무게, 세트 입력
3. 완료한 운동 체크
4. "기록 저장" 버튼 클릭

과거 기록은 달력에서 날짜를 눌러서 확인할 수 있다. 단, 수정은 오늘 날짜만 가능하다.

---

## 마무리

복잡한 기능 없이 딱 필요한 것만 넣었다. 운동할 때 폰 꺼내서 바로 기록하고, 나중에 얼마나 성장했는지 확인하는 용도로 쓰기 좋다.

HTML, CSS, JavaScript만 알면 누구나 이런 앱을 만들 수 있다. 나만의 루틴에 맞게 커스터마이징하는 재미도 있다.
